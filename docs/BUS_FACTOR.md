# Phasmid Continuity Note (Bus Factor)

## Current Maintainer Count

- Maintainer count: `1`

## If Maintainer Becomes Unavailable

- No guaranteed SLA is available.
- Security triage and release cadence may stop without notice.
- If continuity cannot be maintained, the repository may move to explicit EOL status.

## EOL and Archive Conditions

The project may be archived or declared EOL if one or more of the following applies:

- sustained inability to review or merge fixes
- inability to publish security advisories in reasonable time
- inability to validate core security-sensitive changes

## Recommended User Fallback Actions

- Pin to a reviewed commit hash used in your own validation process.
- Maintain an internal fork for critical fixes if operational dependence exists.
- Prefer established alternatives when long-term maintenance guarantees are required.
- Keep local export/recovery procedures documented outside this repository.

## Scope Reminder

Phasmid is a field-evaluation prototype, not a continuity-guaranteed product platform.
